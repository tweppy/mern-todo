# Fullstack Todo App

A simple Fullstack Todo App built with React, Tailwind CSS, DaisyUI, Express, MongoDB, and Mongoose.

## Features

- Add new todo items
- Mark todo items as completed
- Delete todo items

## Technologies Used

- **Frontend**:
  - React
  - Tailwind CSS
  - DaisyUI

- **Backend**:
  - Express
  - MongoDB
  - Mongoose
